How to Use:
	Selecting a cell:
		1. Mouse click

	Entering a value or string:
		1. Input text in "Contents" text box.
		2. Click enter button in GUI or on keyboard.

	Entering a formula:
		1. Enter formula with a preceeding "=".
			a. Value of formula is displayed in cell

Additional Features:
	1. Option to save as a .txt file.
	2. Window displays the name of the file.
	3. Suggests a file name when saving.
	4. Help Menu shows contents of README.
